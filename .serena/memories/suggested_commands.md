Key commands: `./gradlew test` for full tests, `./gradlew test --tests <TestClass>` for focused tests, `./gradlew compileKotlin` for compile-only verification, `./gradlew buildPlugin` to produce the plugin zip under `build/distributions/`, and `./gradlew verifyPlugin` for plugin verification. Standard Darwin shell utilities available include `git`, `ls`, `find`, `sed`, `shasum`, and `rg` for fast searching.
