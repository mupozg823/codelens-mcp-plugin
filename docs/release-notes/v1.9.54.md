# v1.9.54

## Summary

v1.9.54는 `v1.9.53`의 automated harness 위에 **second-pass registry reduction**과
최신 product surface 정리를 얹은 릴리스다.

핵심은 세 가지다.

- compiled source registry를 `113 -> 110 -> 108`로 축소
- deprecated primitive를 visible `tools/list`에서 제거하고 canonical workflow만 노출
- README / docs / D3 architecture page를 최신 runtime surface 기준으로 정합화

## Registry Reduction

이번 릴리스에서 visible registry 기준으로 추가 정리한 항목은 아래 두 개다.

- `get_impact_analysis`
- `find_dead_code`

둘 다 direct call compatibility는 유지한다. 다만 이제는 source registry와
기본 `tools/list`에서 canonical replacement만 보이게 한다.

- `get_impact_analysis` → `impact_report`
- `find_dead_code` → `dead_code_report`

구현은 hidden compatibility definition을 통해 유지한다.
즉, direct call과 schema lookup은 계속 동작하지만, 기본 surface count와
generated manifest는 canonical registry만 반영한다.
