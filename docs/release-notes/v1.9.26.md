# v1.9.26 Release Notes

## Highlights

- **SCIP hover docs in find_symbol**: definitions enriched with documentation from SCIP symbol info
- **SCIP stats in get_capabilities**: `scip_file_count` and `scip_symbol_count` reported when backend is loaded
- **4 tools with SCIP precise path**: find_symbol, find_referencing_symbols, get_file_diagnostics, get_capabilities

## Changes

### find_symbol SCIP Documentation

- SCIP definitions now include `documentation` field from hover docs
- Empty signatures auto-filled from SCIP documentation
- Enriches results beyond what tree-sitter can provide (type info, docstrings)

### get_capabilities SCIP Stats

- `scip_available`: boolean indicating SCIP index presence
- `scip_file_count`: number of indexed files (null if backend not loaded)
- `scip_symbol_count`: number of known symbols (null if backend not loaded)

## SCIP Integration Summary

| Tool                       | SCIP Enhancement                          |
| -------------------------- | ----------------------------------------- |
| `find_symbol`              | Precise definitions + hover documentation |
| `find_referencing_symbols` | Type-aware cross-file references          |
| `get_file_diagnostics`     | Compiler-level diagnostics from index     |
| `get_capabilities`         | Index stats reporting                     |
