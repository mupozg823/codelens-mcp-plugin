# v1.9.23 Release Notes

## Highlights

- **Feature matrix CI verification**: all feature flag combinations (otel, scip-backend, http) verified in quality gate
- **Zero warnings** across all feature configurations (default, no-default, http, otel, scip-backend)
- **SCIP usage guide**: end-to-end documentation for SCIP precise navigation setup
- **Building section expanded**: feature flags table with binary size impact

## Changes

### Feature Matrix Verification

- `scripts/quality-gate.sh` CI mode now verifies all opt-in feature combinations compile cleanly
- Combinations tested: `otel`, `scip-backend`, `http+otel`, `http+scip-backend`

### Warning Cleanup

- `#[cfg(feature = "semantic")]` gating added to `semantic_query_for_embedding_search`, `load_project_bridges`, `bridge_nl_to_code_vocabulary`
- Conditional imports for `HashSet` and `Path` in query_analysis.rs
- Zero compiler warnings across all 6 feature configurations

### Documentation

- `docs/scip-guide.md`: setup instructions for SCIP indexers (Rust, Go, TS, Java, Python), auto-detection paths, integration flow
- README Building section: feature flags table with `semantic`, `http`, `otel`, `scip-backend` descriptions and size impact
- README Enterprise Features: SCIP precise backend entry added
